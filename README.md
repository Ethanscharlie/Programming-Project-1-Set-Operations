# Programming-Project-1-Set-Operations
Demonstrates various operations on sets and multisets. 
To run, 
1. Click on "<> Code" and then click on "Download ZIP".
2. Extract the downloaded .zip folder and keep track of the folder's location.
3. Run operationTest.py in the Command Prompt or chosen Python interpretor.

# operations.py
First, a universal set U is defined.
Part 1 takes two subsets of the universal set U and performs the specified operation on the two subsets. Then, a bitstring/boolean array and the resulting subset is displayed. 
Part 2 takes two multisets of a universal set U and performs the specified operation on the two multisets. Then, a structure (containing the amount of each element in the resulting multiset) and the resulting multiset is displayed.

# operationTest.py
Tests both Parts 1 & 2 of operations.py using unit tests to make sure the functions work correctly.
